Opdracht 3

De tekst in de blokken worden door de class div gecentreerd.

vertical-align: center heeft geen effect omdat de property width niet ingesteld is.

Als men vertical-align: middle gebruikt worden de bolkken op één lijn gezet. De middellijn van alle blokken.

Men kan tekst verticaal centreren door line-height te gebruiken. Als men deze evengroot zet als height dan zal de regel tekst gecentreerd blijven.

Door gebruik te maken van een tabel kan men meerdere regels tekst centreren adhv de align tools.

De class tall krijgt voorang op area. Ondanks dat ze de zelfde specificity hebben, krijgt de class tall voorang omdat deze later gedefineerd is dan de class area.
